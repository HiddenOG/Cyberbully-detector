CyberShield User Guide

Learn how to use the app efficiently and understand its AI-powered features.

Overview

SafeText is an AI-powered platform designed to detect and manage harmful content online. It focuses on cyberbullying, terrorism-related content, and racist remarks.

  • Text Analysis: Check any text for harmful content.
  • Mini Social Feed: Post and comment safely with real-time monitoring.
  • Live Monitoring Feed: Track flagged comments and posts with counters.
  • Chatbot Assistance: Get AI guidance on safe online interactions.
  • Actions Page: Guidance for handling flagged content.

Models Used

  • Toxic Content Detection: unitary/toxic-bert for identifying abusive comments.
  • Detoxification Analysis: Detoxify model for detecting identity-based hate and toxic text.
  • Keyword Lists: Predefined lists for terrorism-related terms and racist slurs for quick detection.

Using SafeText

  1. Home Page: Access `/` to see current stats, charts, and quick links.
  2. Paste Text: Navigate to `/paste` and input text for analysis.
  3. Mini Social Feed: Navigate to `/facebook` to post messages and monitor flagged content.
  4. Live Monitoring Feed: Navigate to `/facebook/live` to watch flagged posts/comments in real-time.
  5. Chatbot: Navigate to `/chatbot` to interact with the AI and get real-time feedback.
  6. Actions: Navigate to `/actions` for recommended steps for flagged content.

Developer Instructions

Follow these steps to run SafeText on your PC:

  1. Install Python (if not already installed). Make sure python --version works in your terminal or command prompt.
  2. Install required packages: Open your terminal/command prompt and run: 
            pip install flask transformers detoxify torch
  3. Start the Flask app: In the terminal, navigate to the folder containing app.py and run: 
            python app.py
  4. Open SafeText in your browser: Go to http://127.0.0.1:5000/. This is the home page.
  5. Use the navbar to navigate between pages like Paste, Chatbot, Facebook, Feed, and Actions. You can always return to Home easily.

Tip: Keep the terminal running while using the app to see logs and live updates.

Tips for Efficient Use

  • Use Paste Text for quick analysis of any message.
  • Post content in Mini Social Feed to see real-time flagging.
  • Use Live Monitoring Feed for multiple posts/comments tracking.
  • Follow Actions Page for recommended steps after harmful content is detected.
